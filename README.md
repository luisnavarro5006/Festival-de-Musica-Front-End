# Festival-de-Musica-Front-End

# Festival C3 

Este proyecto fue realizado con la intención de poder poner aprueba diversos conocimientos, en areas de front-end.

La idea principal de este proyecto era realizar una pgina que intentara imitar una pagina de salon de eventos musicales, en los cuales se presentarian ciertas bandas y demas
la idea general del proyecto era hacer que la pagina se viera de manera correcta y se intentara realizar una buena automatización de proceoss en dado caso de montarse en un servidor
el proyecto a pesar de parecer simple fue realizado con diversos lenguajes de programación web como lo es JS, Gulp (Herramienta de automatización realizada en JS),
CSS, y HTML.

# Parte Ténica
Respecto a la parte técnica el gran reto fue aprender a utilizar Gulp, para la automatización, y el diseño responsive del sitio web, fue realizado con SASS, HTML y JS

Se utilizaron diversos formatos de multimedia para lograr que la pagina fuera mucho mas ligera que otras de mi autoria, en este caso utilizamos el formato .webp para que el video de la pagina 
principal no necesitara mucho espacio al igual que los diversos logos e imagenes.

Tambien, es un hecho de que gracias a Gulp, se pudo realizar un diseño responsive para poder utilizarse en cualquier dispositivo, desde Laptops, Monitores Grandes, Tablets, Smartphones entre otras cosas.

# Conclusión
Es una pagina web bastante sencilla, pero a pesar de todo eso tiene bastante empeño detras ya que el hecho de aprender simplemente a manejar SASS junto a Gulp ya es un dolor de cabeza,
de alguna manera podemos decir que esta pagina, puede ser la base de muchas otras teniendo en cuanta que se puede reutilizar bastante codigo como hueso para cualquier otro proyecto
ya que ninguno de los efectos visuales esta hecho con algun otro framework como lo es bootstrap.
